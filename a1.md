Platform:

* Flutter (Android + iOS)

---

1. PRODUCT VISION

---

SecureVault is a premium password and document manager that allows users to securely store passwords, backup codes, and personal documents using military-grade encryption.

Primary Goal:

Even if:

* Device is stolen
* Database is leaked
* Exported backup file is stolen
* Cloud backup is compromised

Attackers must NOT be able to read user passwords or documents without the correct passphrase.

The application follows a Zero-Knowledge model.

6. SECURITY MODEL

---

Master Password

User creates master password on first launch.

Master password is NEVER stored.

Storage:

Argon2id Hash

Example

Master Password
↓
Argon2id
↓
Stored Hash

Verification:

Entered Password
↓
Argon2id
↓
Compare Hash

7. ENCRYPTION

---

Algorithm:

AES-256-GCM

Key Derivation:

Argon2id

Every Password Record

Unique Salt

Master Password
↓
Argon2id
↓
AES Key
↓
Encrypt Record

Stored:

cipherText
iv
salt
tag

No plaintext password should exist in database.
